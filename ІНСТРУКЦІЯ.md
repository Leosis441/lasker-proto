# Як побачити прев'ю сайту

## Крок 1: Встановіть Node.js

Якщо у вас macOS, найпростіше встановити через Homebrew:

```bash
brew install node
```

Або завантажте з офіційного сайту:
https://nodejs.org/

## Крок 2: Встановіть залежності проєкту

Відкрийте термінал у папці проєкту і виконайте:

```bash
cd "/Users/cieden1/Documents/Lasker Proto"
npm install
```

## Крок 3: Запустіть сервер розробки

```bash
npm run dev
```

## Крок 4: Відкрийте браузер

Після запуску ви побачите повідомлення типу:
```
✓ Ready in 2.3s
○ Local:        http://localhost:3000
```

Відкрийте браузер і перейдіть за адресою: **http://localhost:3000**

---

## Альтернатива: Використання VS Code

Якщо ви використовуєте VS Code:
1. Відкрийте папку проєкту в VS Code
2. Натисніть `Cmd + Shift + P` (Mac) або `Ctrl + Shift + P` (Windows)
3. Введіть "Terminal: Create New Terminal"
4. Виконайте команди з кроків 2-3 вище

---

## Що ви побачите

Після відкриття http://localhost:3000 ви побачите:
- **Ліва частина**: Темна панель з ефектом свічення та текстом про Lasker AI
- **Права частина**: Форма входу з полями для email та пароля

Ефект свічення на лівій панелі активується при наведенні миші!
