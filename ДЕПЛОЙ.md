# Як опублікувати прототип онлайн (Vercel)

## Варіант 1: Через Vercel (Найпростіший - 5 хвилин) ⭐

### Крок 1: Зареєструйтеся на Vercel
1. Перейдіть на https://vercel.com
2. Натисніть "Sign Up"
3. Увійдіть через GitHub (рекомендовано) або email

### Крок 2: Встановіть Vercel CLI (опціонально, але простіше через веб-інтерфейс)

**Через веб-інтерфейс (НАЙПРОСТІШЕ):**
1. Перейдіть на https://vercel.com/new
2. Натисніть "Import Git Repository"
3. Якщо проєкт ще не на GitHub:
   - Створіть репозиторій на GitHub
   - Завантажте файли проєкту туди
4. Виберіть ваш репозиторій
5. Vercel автоматично визначить Next.js
6. Натисніть "Deploy"
7. Зачекайте 2-3 хвилини
8. Готово! Ви отримаєте посилання типу: `https://your-project.vercel.app`

**Через CLI (якщо встановлено Node.js):**
```bash
npm install -g vercel
vercel login
vercel
```
Слідуйте інструкціям на екрані.

## Варіант 2: Через Netlify

1. Перейдіть на https://www.netlify.com
2. Зареєструйтеся
3. Натисніть "Add new site" → "Import an existing project"
4. Підключіть GitHub репозиторій
5. Налаштування:
   - Build command: `npm run build`
   - Publish directory: `.next`
6. Натисніть "Deploy"

---

## Варіант 3: Статичний експорт (для простих прототипів)

Якщо потрібен статичний сайт без сервера:

1. Оновіть `next.config.js`:
```javascript
const nextConfig = {
  output: 'export',
  images: {
    unoptimized: true
  }
}
```

2. Зберіть статичні файли:
```bash
npm run build
```

3. Завантажте папку `out` на будь-який хостинг (GitHub Pages, Netlify Drop, тощо)

## Після деплою

✅ Ви отримаєте посилання типу: `https://your-project.vercel.app`
✅ Це посилання можна надсилати іншим
✅ Зміни автоматично оновлюються при push в GitHub (якщо використовуєте GitHub)

---

## Поради

- **Безкоштовний план Vercel** дає вам:
  - Необмежену кількість проєктів
  - Автоматичні HTTPS
  - Глобальний CDN
  - Автоматичні деплої з GitHub

- **Для приватного прототипу**: Vercel дозволяє зробити проєкт приватним (тільки для вас)

- **Кастомний домен**: Можна підключити свій домен безкоштовно
