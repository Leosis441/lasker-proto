# 📍 Куди саме завантажувати файли на GitHub

## Крок 1: Відкрийте ваш репозиторій

1. **Увійдіть на GitHub.com** (якщо ще не увійшли)

2. **Знайдіть ваш репозиторій:**
   - Натисніть на ваш аватар (фото) в правому верхньому куті
   - Виберіть "Your repositories" (Ваші репозиторії)
   - Або перейдіть за адресою: `https://github.com/ВАШЕ_ІМ'Я/lasker-proto`
     (замініть ВАШЕ_ІМ'Я на ваше ім'я користувача GitHub)

3. **Відкрийте репозиторій** `lasker-proto` (або як ви його назвали)

---

## Крок 2: Знайдіть місце для завантаження

### Варіант А: Якщо репозиторій ПУСТИЙ (тільки що створений)

Ви побачите сторінку з текстом типу:
```
Quick setup — if you've done this kind of thing before
```

**Прокрутіть сторінку ВНИЗ** до секції:

```
…or create a new repository on the command line
…or push an existing repository from the command line
…or upload files
```

**Натисніть на "upload files"** (завантажити файли)

---

### Варіант Б: Якщо репозиторій вже має файли

1. **Натисніть кнопку "Add file"** (Додати файл) в правому верхньому куті
2. **Виберіть "Upload files"** (Завантажити файли)

---

## Крок 3: Завантажте файли

### Що ви побачите:

Велике поле з текстом:
```
Drag files here to add them to your repository
or choose your files
```

### Що робити:

**Спосіб 1: Перетягнути файли (Drag & Drop)**

1. **Відкрийте Finder** (на Mac) або Провідник (на Windows)
2. **Перейдіть до папки:** `/Users/cieden1/Documents/Lasker Proto`
3. **Виберіть ВСІ файли та папки:**
   - Натисніть `Cmd + A` (Mac) або `Ctrl + A` (Windows) щоб вибрати все
   - АБО виберіть вручну:
     - Папку `app`
     - Папку `components`
     - Папку `lib`
     - Файл `package.json`
     - Файл `tsconfig.json`
     - Файл `tailwind.config.ts`
     - Файл `next.config.js`
     - Файл `postcss.config.js`
     - Файл `components.json`
     - Файл `.gitignore`
     - Файл `README.md`
     - Всі інші файли (крім папки `.next` якщо вона є)

4. **Перетягніть файли** в браузер на поле "Drag files here"

5. **Зачекайте**, поки файли завантажаться (побачите прогрес-бар)

---

**Спосіб 2: Кнопка "choose your files"**

1. **Натисніть "choose your files"** (вибрати файли)
2. **Відкриється вікно вибору файлів**
3. **Перейдіть до папки:** `/Users/cieden1/Documents/Lasker Proto`
4. **Виберіть ВСІ файли та папки** (як описано вище)
5. **Натисніть "Open"** (Відкрити)

---

## Крок 4: Збережіть зміни

1. **Прокрутіть сторінку ВНИЗ**

2. **Знайдіть поле "Commit changes"** (Зберегти зміни)

3. **Введіть повідомлення** (необов'язково, але корисно):
   ```
   Initial commit - Lasker AI prototype
   ```

4. **Натисніть зелений кнопку "Commit changes"** (Зберегти зміни)

5. **Зачекайте 10-30 секунд** поки файли завантажаються

---

## Крок 5: Перевірте результат

Після завантаження ви побачите:

✅ Список файлів та папок:
   - `app/`
   - `components/`
   - `lib/`
   - `package.json`
   - `tsconfig.json`
   - і т.д.

**Якщо ви бачите всі файли - все готово! ✅**

---

## ❓ Часті питання:

### Я не бачу кнопку "upload files"
**Рішення:** 
- Переконайтеся, що ви на сторінці ВАШОГО репозиторію
- Спробуйте оновити сторінку (F5 або Cmd+R)
- Переконайтеся, що репозиторій створено як **Public**

### Файли не завантажуються
**Рішення:**
- Перевірте інтернет-з'єднання
- Спробуйте завантажити менше файлів за раз
- Спробуйте інший браузер

### Я бачу тільки деякі файли
**Рішення:**
- Переконайтеся, що ви вибрали ВСІ файли та папки
- Папки (`app`, `components`, `lib`) теж потрібно завантажити
- Всередині папок теж мають бути файли
