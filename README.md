# Lasker AI Prototype

A clickable prototype for Lasker AI built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui components.

## Features

- Dark mode UI components from shadcn/ui
- Glowing effect component from Aceternity UI
- Responsive design
- TypeScript for type safety
- Ready for deployment to Vercel/Netlify

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy to Production

### Quick Deploy to Vercel (Recommended)

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com) and sign in
3. Click "Import Project" and select your GitHub repository
4. Vercel will auto-detect Next.js settings
5. Click "Deploy"
6. Get your live URL (e.g., `https://your-project.vercel.app`)

**Detailed instructions:** See `ДЕПЛОЙ.md` (Ukrainian) for step-by-step guide.

## Project Structure

- `/app` - Next.js app directory with pages
- `/components/ui` - shadcn/ui components and custom components
- `/lib` - Utility functions

## Components Used

- shadcn/ui components (Button, Input, Label, Separator)
- Aceternity UI Glowing Effect component
- Dark mode styling throughout
